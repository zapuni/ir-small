## 5. Context Preservation Example

Original: "Machine Learning là nhánh của AI. Deep Learning sử dụng neural 
          networks. RAG kết hợp retrieval và generation."

WITHOUT overlap:
┌────────────────────────────────┐
│ Chunk 1: "ML là nhánh của AI.  │
│ Deep Learning..."              │
└────────────────────────────────┘
                                    ← Gap!
┌────────────────────────────────┐
│ Chunk 2: "RAG kết hợp..."      │  ← Lost context về Deep Learning!
└────────────────────────────────┘

WITH 27% overlap:
┌────────────────────────────────┐
│ Chunk 1: "ML là nhánh của AI.  │
│ Deep Learning sử dụng..."      │
└────────────┬───────────────────┘
             │ Overlap: "Deep Learning sử dụng neural networks."
             ↓
┌────────────┴───────────────────┐
│ Chunk 2: "Deep Learning...     │  ← Context preserved! ✅
│ RAG kết hợp..."                │
└────────────────────────────────┘

## 6. Query Matching Visualization

Query: "Deep Learning hoạt động như thế nào?"

Embedding: [0.2, 0.8, 0.5, ...]

┌─────────────────────────────────┐
│ Chunk 1                         │
│ "ML là nhánh của AI.            │  Similarity: 0.72 ✅
│ Deep Learning sử dụng NN..."   │  → Matches "Deep Learning"
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ Chunk 2                         │
│ "Deep Learning sử dụng NN.      │  Similarity: 0.85 ✅✅
│ NN có nhiều layers..."          │  → Best match! Có detail
└─────────────────────────────────┘

┌─────────────────────────────────┐
│ Chunk 3                         │
│ "RAG kết hợp retrieval..."      │  Similarity: 0.35
└─────────────────────────────────┘  → Low relevance

Result: Retrieve Chunk 2 and Chunk 1 (both contain answer!)
